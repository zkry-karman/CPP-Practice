**Description**

The exercises of this project serve to teach an introdution to the C++ language. Its comprised of 3 exercises which ask to create simple programs to show how C++ differs from its roots, C. 

**Instructions**

Upon cloning this project, each exercise is compiled seperately from the others. Open the exercise you want to look at, and run 'make'. The executable file for each of the projects does not take any additional files or parameters. Each program has a main that can be used to execute various tests within the program.
