**Description**

The exercises of this project serve to teach an introdution to the C++ language. Its comprised of 3 exercises which ask to create simple programs to show how C++ differs from its roots, C. This module is composed of 6 exercises, all adding onto the previous one. The lessons that can be taken away from this module is how to create instances of a class, and call its functions, and see how memory is stored and managed in cpp. 

**Instructions**

For each exercise of this module, you just need to compile with <make> and run the executable file. No additional parameters needed for execution. The main() for each exercise already has it's own tests within the program, but feel free to add your own to see how it works :')
